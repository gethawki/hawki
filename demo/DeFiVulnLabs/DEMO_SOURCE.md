# Demo source

Vendored from https://github.com/SunWeb3Sec/DeFiVulnLabs (SunWeb3Sec), a corpus
of self-contained Foundry reproductions of real, documented on-chain exploit
patterns (48+ vulnerability types with incident references). Used as the 4th
Hawk-i demo to prove the WHOLE pipeline end to end on real-incident code: the
50 static rules AND the autonomous deep agent (known rule-attacks -> novel
LLM-invented attack -> live sandbox execution).
