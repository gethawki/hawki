# Demo source

Vendored from https://github.com/pancakeswap/pancake-smart-contracts, the public
monorepo for PancakeSwap, the largest DEX/AMM on BNB Chain. This demo scans the
farms-pools project (the real MasterChef/CakeToken/SmartChef farming core that is
deployed on BNB Chain) to exercise Hawk-i against production-grade, audited,
major-protocol code on a major non-Ethereum chain.
